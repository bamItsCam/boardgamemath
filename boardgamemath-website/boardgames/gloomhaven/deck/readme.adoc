== Converting PDF to images

For example, the CH cards start from card number `146`:

----
convert CH\ Cards.pdf CH.png
rm *1.png *3.png *5.png *7.png *9.png
i=146; for file in `ls *.png | sort -V`; do mv $file CH-$i.png; i=$((i+1)); done
----
